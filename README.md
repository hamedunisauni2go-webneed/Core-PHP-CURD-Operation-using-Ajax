# Core PHP CRUD Operation using AJAX

## 📌 Project Overview
This is a simple CRUD (Create, Read, Update, Delete) application developed using Core PHP with AJAX. The project performs all database operations without page reload, providing a smooth and dynamic user experience.

---

## 🚀 Features
- Add new records (Create)
- Display records (Read)
- Update existing records (Update)
- Delete records (Delete)
- No page reload using AJAX
- Responsive and simple UI

---

## 🛠️ Technologies Used
- PHP (Core PHP)
- MySQL Database
- HTML5
- CSS3
- JavaScript
- jQuery AJAX

---

## ⚙️ How It Works
- User interacts with the UI
- AJAX sends requests to PHP backend
- PHP processes data and interacts with MySQL
- Response is returned without page reload

---

## 📂 Project Structure
- index.php → Main file (UI)
- users.sql → table import
-  Insert data logic
-  Update data logic
-  Delete data logic
-  Fetch records data logic
-  All above data logic done on single page index.php
